## The Effect of Source Prestige on the Perceived Credibility of Health Information
This repository contains the published HTML version of my PSY 120L final reproducible report.

Does the prestige of a source influence the percieved credibility of information, and does this effect depend on the participants' need for cognition or trust in science?

This report was created in Quarto/R. The report tile in index.html The dataset is not included in this repository.
